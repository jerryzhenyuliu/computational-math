# Computational Math Explorations

Python scripts exploring classical problems in number theory and numerical
analysis. Standard library only — no NumPy, no SciPy.

## What's in so far

- [`01_pi_monte_carlo.py`](01_pi_monte_carlo.py) — estimate pi by dropping random points, watch the `1/sqrt(N)` error rate
- [`02_e_three_ways.py`](02_e_three_ways.py) — three ways to approach e: random-sum, derangements, and Taylor series
- [`03_primes.py`](03_primes.py) — Sieve of Eratosthenes + Prime Number Theorem + gap distribution + Bertrand's postulate
